# DoseCerta-Web

Aplicação **front-end em React** do projeto **DoseCerta**, desenvolvida para apoiar a gestão e visualização de informações relacionadas à vacinação, promovendo organização, eficiência e melhor tomada de decisão na área da saúde.

---

## Visão Geral

O **DoseCerta-Web** é a interface web que consome a API do projeto DoseCerta, permitindo o gerenciamento de dados como unidades de saúde, profissionais e demais informações necessárias ao controle de vacinas.

---

## Tecnologias Utilizadas

* **React**
* **JavaScript (ES6+)**
* **Axios** – consumo da API
* **Node.js / npm**

---

##  Como Executar o Projeto

### Pré-requisitos

* Node.js instalado
* npm ou yarn

### Passos

```bash
# Clonar o repositório
git clone https://github.com/seu-usuario/DoseCerta-Web.git

# Acessar a pasta do projeto
cd DoseCerta-Web

# Instalar as dependências
npm install

# Iniciar a aplicação
npm start
```

A aplicação estará disponível em:

```
http://localhost:3000
```
